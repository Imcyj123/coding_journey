### 內容
這個簡單的 C++ 函數會回傳什麼值呢？ 

long long H(int n){

long long res = 0;

for( int i = 1; i <= n; i=i+1 ){

res = (res + n/i);

}

return res;

}


### 輸入說明
輸入的第一行是一個整數 T ( T <= 1000 )，表示有幾個測試資料。接下來的 T 行每行有一個 32 位元的有號整數 n。

### 輸出說明
對於每個測資，輸出 H(n) 於一行。

- 範例輸入 #1
```
2
5
10
```

- 範例輸出 #1
```
10
27
```
